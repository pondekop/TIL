# 📋 Markdown Cheat Sheet!

## 1. 見出し (Headings)

# 見出し1 (H1)
## 見出し2 (H2)
### 見出し3 (H3)

---

## 2. 改行(Line break)
文末で半角スペースを  
2つ入力すると改行される

## 3. 強調 (Emphasis)

*斜体*  
**太字**  
***斜体&太字***  
~~取り消し線~~

---

## 4. リスト (Lists)

### 4.1. 番号なしリスト (Unordered list)

- text1
  - text1-1
- text2

### 4.2. 番号付きリスト (Ordered list)

1. text1
2. text2
   1. text2-1
   2. text2-2

---
